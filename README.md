# To-Do List App

A simple command-line to-do list application in Python. Tasks are saved to a file so you can keep your list between runs.

## Features
- Add tasks
- View tasks
- Remove tasks
- Tasks are saved in a file (`tasks.txt`)

## How to Run
1. Make sure you have Python 3 installed.
2. Open a terminal in this folder.
3. Run:
   ```
   python todo.py
   ```

## Example
```
To-Do List App
1. Add Task
2. View Tasks
3. Remove Task
4. Exit
Enter your choice: 1
Enter task: Buy groceries
Task added.
```
